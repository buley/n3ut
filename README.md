# n3ut

A blockchain-based mediation tool built on Echo and Vue.js, and based on [@m0t0k1ch1's login example][https://github.com/m0t0k1ch1/metamask-login-sample/].

## Requirements

* Node.js
* Yarn
* Golang

## Install

``` sh
$ yarn install
```

## Build

``` sh
$ go build
$ yarn build
```

## Run

```
$ ./n3ut --conf _config/sample.json
```
